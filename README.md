# cross-video-encoder
A lightweight, robust, and cross-platform video encoder built with Python, Tkinter, and FFmpeg. It provides a simple graphical interface to quickly compress videos using optimized $H.264$ presets across Linux, Windows, and macOS.

## Features

* **Cross-Platform:** Native execution on Linux, Windows, and macOS.
* **Streamlined UI:** Built with Tkinter (`main.py`) for a lightweight, dependency-free graphical interface.
* **Robust Core:** Powered by FFmpeg for efficient and reliable video processing.
* **3 Optimized Presets:** Pre-configured H.264 compression levels (Fast/Low, Balanced/Medium, High Quality) to balance speed and file size without complexity.
* **No Bloat:** Simple, effective, and focused strictly on reliable encoding.

## Requirements

Before running the application, ensure you have the following prerequisites installed and configured:

1. **Python 3.x**
2. **FFmpeg** (Must be added to your system's `PATH`)
